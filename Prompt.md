# MASTER PROMPT — Build a Premium Interactive Portfolio Website

## Objective

Create a premium-quality personal portfolio website that looks like it was designed by an award-winning creative developer, not an AI-generated template.

The design philosophy should combine minimalism, cinematic motion, editorial typography, and smooth 2D interactions while remaining extremely professional.

The website should impress:

- Recruiters

- Hiring managers

- Startup founders

- Open Source Maintainers

- Hackathon judges

- Technical interviewers

The goal is to make visitors immediately think:

> "This developer cares about quality."


# Theme

Primary Theme:

Black & White

Accent:

Only grayscale.

NO bright colors.

Instead use:

- White

- Black

- Multiple grays

- Very subtle glass effects

- Soft shadows

The website should include a Theme Inversion Toggle.

Instead of Light/Dark mode, create an **Invert Mode**

Default:

White background  
Black text

Toggle:

Black background  
White text

All illustrations, SVGs, icons and animations should smoothly transition.

No flashing.

Transition duration:

500ms–700ms.

# Final Goal

The finished website should feel handcrafted, cinematic, memorable, and premium. It should immediately communicate technical excellence and design maturity while remaining easy to maintain. Every section must be editable through structured data, animations should be purposeful rather than distracting, and the overall experience should leave recruiters and visitors with the impression that this portfolio was custom-designed specifically for its owner rather than generated from a template.
